# NOXE - Premium Streaming UI

A modern, cinematic movie streaming interface built with React, Vite, and Tailwind CSS.

## Features
- 🎬 **Cinematic Hero Section**: Dynamic layouts with rich metadata.
- 📱 **Responsive Design**: Optimized for all devices.
- 🔍 **Interactive Components**: Search bars, hover effects, and play buttons.
- 🏷️ **Genre Filtering**: Dynamic category navigation.
- ⚡ **Vite Powered**: Ultra-fast development and build times.

## Tech Stack
- **Frontend**: React 19
- **Styling**: Tailwind CSS 4
- **Icons**: React Icons
- **Routing**: React Router 7

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
4. Build for production: `npm run build`
